# Node.js Fastify bearer app

This simple app utilizes fastify bearer with nodejs and to showcase the use of the plugin.

## Getting Started

Follow these steps to get your development environment set up:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run linting:

   ```bash
   npm run lint
   ```

3. Run tests:

   ```bash
   npm test
   ```

4. Start the server:
   ```bash
   npm start
   ```
